# nextest Skill

This repository contains a reusable agent skill that provides guidance on
using `cargo-nextest`, the next-generation test runner for Rust.

## Installation

Install the skill by copying the `nextest` directory into your agent skills directory:

- Codex: `~/.codex/skills`
- Claude Code: `~/.claude/skills`

For example:

```bash
cp -R nextest ~/.codex/skills/
cp -R nextest ~/.claude/skills/
```
